# Git Workflow

## Push to main

- You **cannot** push directly to the `main` branch without confirmation. Ask the user first.
- Always ensure changes are correct.

## Branch creation

- If you want to create a new branch with your changes, **ask** the user first before creating it.
- Wait for user confirmation before creating any new branch.
